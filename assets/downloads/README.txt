# Dossier Téléchargements

Ce dossier contient les documents téléchargeables du portfolio :

1. cv.pdf - CV détaillé (à ajouter)
2. exemple-cdc.pdf - Exemple de cahier des charges (à ajouter)
3. template-roadmap.pdf - Template de roadmap projet (à ajouter)

## Instructions pour ajouter les fichiers :

1. Créez vos documents PDF
2. Placez-les dans ce dossier avec les noms exactes mentionnés
3. Les liens de téléchargement fonctionneront automatiquement

## Sécurité :

- Les téléchargements sont gérés par le serveur web
- Aucune exécution de code n'est possible
- Seuls les fichiers PDF sont recommandés
